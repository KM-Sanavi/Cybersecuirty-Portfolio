# Personal Security Audit

## Overview
This project documents a personal digital security assessment conducted to evaluate the security posture of my online accounts, devices, and public OSINT exposure.

## Objectives
- Assess account security settings.
- Identify security risks and misconfigurations.
- Review OSINT exposure.
- Document findings based on severity.
- Recommend and implement remediation measures.

## Methodology
Scope → Intelligence Gathering → Exposure Assessment → Severity Classification → Remediation

## Tools Used
- Google Security Checkup
- Have I Been Pwned
- EPIEOS (OSINT)
- GitHub Security Settings
- LinkedIn Privacy Settings
- Apple ID Security Settings

## Key Findings
- Multi-factor Authentication (MFA) was disabled on several accounts.
- Unused third-party applications had access to the Google account.
- Public OSINT exposure was identified.
- Default router administrator password had not been changed.

## Actions Taken
- Enabled MFA on supported accounts.
- Revoked unused third-party application access.
- Reviewed account privacy settings.
- Verified password strength and uniqueness.

- Security Best Practices
